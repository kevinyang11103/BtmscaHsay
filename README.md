# 课程管理系统 java1186

## 项目概述

课程管理系统是一个基于B/S架构的Web应用程序，采用MVC模式设计，实现前后端分离。系统致力于提供便捷的课程管理功能，服务于管理员、教师和学生。

## 技术栈

- **后端：** Java、SpringBoot
- **前端：** Vue
- **数据库：** Mysql
- **开发环境：** IDEA、JDK1.8、Maven
- **系统环境：** Windows

## 功能模块

### 系统角色

- 管理员
- 教师
- 学生

### 管理员模块

- 首页
- 系管理
- 专业管理
- 办理管理
- 学生管理
- 教师管理
- 课程管理
- 选课修改
- 管理员管理

### 教师模块

- 首页
- 授课查询
- 教师课表
- 成绩查询

### 学生模块

- 首页
- 选修课程
- 学生课程
- 课表查询
- 成绩查询
- 信息维护
